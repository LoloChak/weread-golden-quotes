# 微信读书每日金句生成器

从你的微信读书笔记中提取高质量金句，生成宣纸质感人文风格网页，每日随机展示 + 思考问题引导。

---

## 功能特色

| 功能 | 说明 |
|------|------|
| 智能筛选 | 22 组洞察性关键词过滤，自动识别金句 |
| 思考问题 | 每条金句附带基于内容生成的思考问题 |
| 每日金句 | 基于日期种子的确定性随机，同一天看到同样的金句 |
| 换一批/再来几条 | 随机刷新，去重、去同书 |
| 收藏功能 | localStorage 持久化，支持查看、删除、导出 TXT |
| 宣纸质感 | 京华老宋体 + LXGW文楷 + SVG 噪点底纹 |
| 分类颜色 | 12 个分类暖色调配色，自动识别内容分类 |
| 响应式 | 移动端适配 |

## 快速开始

### 1. 获取 API Key

1. 打开微信读书 App
2. 进入「我」→「设置」→「账户与安全」
3. 找到「Agent API Key」并生成（格式 `wrk-xxxxxxxx`）

### 2. 配置环境变量

```bash
export WEREAD_API_KEY="wrk-xxxxxxxx"
```

### 3. 安装依赖

```bash
pip install requests
```

### 4. 生成金句

```bash
cd weread-golden-quotes
python scripts/fetch_quotes.py -o ./output
```

生成的文件：

```
output/
├── quotes_data.js     # 金句数据（供 HTML 引用）
└── 每日金句.html       # 打开即可使用的金句网页
```

## 命令参数

```bash
python scripts/fetch_quotes.py [选项]
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-o, --output` | 输出目录 | `./output` |
| `--max-books` | 最多处理多少本书 | 全部 |
| `--json` | 同时输出 JSON 格式 | 关闭 |
| `--template` | 自定义 HTML 模板路径 | 内置模板 |
| `--no-html` | 不生成 HTML（只输出数据） | 关闭 |

### 示例

```bash
# 快速测试：只处理前 20 本书
python scripts/fetch_quotes.py --max-books 20

# 同时输出 JSON（供二次开发）
python scripts/fetch_quotes.py --json

# 使用自定义模板
python scripts/fetch_quotes.py --template ./my_template.html

# 只生成数据，不生成 HTML
python scripts/fetch_quotes.py --no-html
```

## 数据结构

每条金句的数据结构：

```json
{
  "t": "真正的自由不是想做什么就做什么，而是不想做什么就可以不做什么",
  "b": "1000个铁粉",
  "a": "某某某",
  "c": "个人成长",
  "y": "highlight",
  "q": "真正的自由在你的生活中意味着什么？"
}
```

| 字段 | 说明 |
|------|------|
| `t` | 金句文本 |
| `b` | 书名 |
| `a` | 作者 |
| `c` | 分类（文学/个人成长/艺术/经济理财/心理/教育学习/历史/哲学宗教/科学技术/社会文化等） |
| `y` | 类型：`highlight`（划线）或 `thought`（想法） |
| `q` | 思考问题 |

## 质量筛选规则

1. **长度筛选**：15-150 字
2. **关键词过滤**：22 组洞察性关键词（不是/而是/本质/真正/关键/核心/根本/唯一/最重要/秘诀/真相/底层/逻辑/规律/原则/误区/陷阱/反直觉/洞察/发现/秘密/定律）
3. **内容清洗**：排除纯标点、章节标题、引用标记
4. **前缀去重**：同一来源的相似金句只保留一条

## 思考问题生成

- 匹配到洞察关键词时，使用对应的 2 套问题模板随机选择
- 未匹配时，从 5 条通用问题中随机选择
- 共 49 条专属问题 + 5 条通用问题

## 自定义模板

如果你想修改页面样式，可以复制 `assets/template.html` 并修改，然后通过 `--template` 参数指定。

模板关键约定：
- 必须引用 `<script src="quotes_data.js"></script>`
- 数据通过全局变量 `QUOTES` 访问
- 统计数字由 JS 动态计算，无需手动修改
- 收藏使用 `localStorage` 键 `wr_fav_list`

## 注意事项

| 项目 | 说明 |
|------|------|
| API Key | 通过环境变量 `WEREAD_API_KEY` 传递，请勿提交到公开仓库 |
| 请求频率 | 脚本内置 0.2-0.3s 请求间隔，避免触发限流 |
| 首次运行 | 笔记较多的用户可能需要几分钟 |
| 网络要求 | 生成时需联网获取数据；HTML 页面需网络加载字体 CDN |
| 数据安全 | 所有数据仅在本地处理，不上传任何信息 |
